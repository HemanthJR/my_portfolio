Student Management System
  • The Student Management System is a project developed using HTML, CSS, and JavaScript for the front-end, with Django as the framework and Python and SQLite for the back-end. This system is primarily focused on monitoring the regular activities of school students.
  • The existing system operates manually, making record searching a long process and lacking security measures, which poses a risk of data loss without any recovery options. In contrast, the new system functions as a website offering enhanced security, fast record searching, and secure data storage in the cloud.
  • The system comprises three modules: admin, staff, and student. The admin module plays a crucial role in managing all records.
  • he staff module includes features such as assigning marks to students, managing attendance, applying for leave, and providing feedback.
  • The student module allows students to view their marks and attendance, apply for leave, and give feedback.
  • Additionally, the proposed system includes language options to accommodate students' parents who may not be well-versed in English, with available translations in Tamil and other languages.
  • Furthermore, the system has been developed as a mobile application for ease of use.

The Martin | Mini - Project
  • The name of the project is The Martin.
  • To develop this project, I have used HTML, CSS, JavaScript, Flask, and MongoDB
  • The project is named "The Martin" because it revolves around the concept of buying land on the moon.
  • The name reflects the futuristic and imaginative nature of the project, aiming to capture the curiosity and interest of users who dream of owning a piece of lunar real estate.
  • It is a simple project that takes input from a user, stores it in a database, and displays it on the same page.
  • Specifically, users can enter details such as their name and the desired plot on the moon. This information is then stored securely in a MongoDB database.
  • Once the data is stored, the same page updates to display the entered information, providing instant feedback to the user and confirming their "purchase" of lunar land.

Moviepedia | Mini - Project
  • The name of the project is Moviepedia. In this project, I have used web scraping techniques to extract elements, tags, and other data from web pages, and display them in a structured template.
  • Specifically, I targeted the IMDb website to scrape data from their top 250 movies list. From the top 250 movies, I selected 50 movies to display on the Moviepedia website.
  • To perform the web scraping, I used a Python package called BeautifulSoup (bs4).
  • The extracted data includes movie titles, ratings, directors, and other relevant details.
  • Users can visit the Moviepedia website, browse through the selected 50 movies, and view detailed information about each movie.Additionally, users can select any movie from the list and post comments, including star ratings and textual comments.
  • All user comments and ratings are stored in a MongoDB server to ensure data persistence and easy retrieval.
  • The system allows users to interact with the content dynamically, enhancing their browsing experience.
